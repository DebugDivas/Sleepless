About the product

Sleepless is a cutting-edge safety solution designed to detect drowsiness in drivers and prevent potential accidents on the road. Sleepless continuously monitors the driver's behavior, including eye movements and eyelid patterns. When signs of drowsiness are detected, the system activates immediate alerts, such as visual cues and audible alarms, to jolt the driver back to alertness.

F.A.Q

  • How is this site made?

    This site was created using html and css.

  • How does the code detect drowsiness?

    The code uses facial landmarks to track eye closure, mouth openness (yawning), and head tilt. It calculates metrics like Eye Aspect Ratio (EAR) and Mouth Aspect Ratio (MAR) to detect blinks, prolonged eye closure, yawning, and monitors head tilt to signal potential drowsiness.

  • What are the main functions of this code for facial analysis?

    This code analyzes faces in real-time from a webcam feed. It monitors eye closure, blinks, yawning, and head tilt. It detects multiple faces, tracks eye and mouth movements, and alerts for potential drowsiness based on predefined thresholds.

  • Can the code detect multiple faces simultaneously?

    Yes, the code is capable of detecting and analyzing multiple faces in the camera's view. It identifies each face's facial landmarks independently to monitor individual eye movements, blinks, yawning, and head tilt.

  • How does the code identify yawning in individuals?

    The code employs Mouth Aspect Ratio (MAR) to detect yawning. By measuring the ratio of distances between mouth landmarks, it identifies when this ratio surpasses a predefined threshold, indicating a yawn. An alert is triggered upon detecting extended mouth openness beyond the set threshold.

  • What action triggers an alert for potential drowsiness regarding head tilt?

    The code monitors the degree of head tilt. If the measured head tilt exceeds specific predefined thresholds for a set duration, typically between 10 to 50 degrees for more than 3 seconds, it triggers an alert indicating potential drowsiness due to prolonged head tilt.
